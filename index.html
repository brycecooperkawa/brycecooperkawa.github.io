Volkswagen Shopping App with Augmented Reality

An application I created as part of a team of developers during the MSU Computer Science Capstone Course for our client Volkswagen. The application allows users to customize their own Volkswagen vehicles with different colors and accessories. Users can then display the vehicle in full size wherever they would like. The backend of the application contains user information and model files. The backend can be accessed via the Admin Panel (see below), which allows the model files to be added to, edited or deleted without having to access the database directly.

Traffic Jam Assist System

A traffic jam assist software system developed as part of a group in CSE 435 Software Engineering. This system would exist within an automotive vehicle and would allow the driver to set a specific distance that the vehicle would maintain. The software allows the user's vehicle to operate while in traffic. The SRS document provides requirements, diagrams, constraints, and models that provide all information necessary about the system and its functions.
